# Website media

Managed from the website's admin panel at `/admin/` — no need to touch this folder
directly (though uploading via GitHub's web UI works too).

- `hero/` — full-screen homepage slider photos (wide/landscape images look best)
- `gallery/` — photos & short video clips for the "From the Wings" gallery

Filenames become captions: `andha-yug-act-2.jpg` → "Andha yug act 2".
Images: .jpg .png .webp .avif · Videos: .mp4 .webm .m4v (keep under ~25 MB).

The `demo-*` files in hero/ and the sample file in gallery/ are placeholders —
delete them from the admin panel once real production photos are uploaded.
