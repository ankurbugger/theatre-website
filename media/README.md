# Gallery media

Drop your production photos and video clips in this folder — the website gallery
picks them up automatically. No code changes needed.

- Images: .jpg .jpeg .png .webp .gif .avif
- Videos: .mp4 .webm .m4v (keep under ~50 MB; GitHub blocks files over 100 MB)
- The filename becomes the caption: `andha-yug-act-2.jpg` → "Andha yug act 2"

Easiest upload: open this folder on github.com → "Add file" → "Upload files" →
drag & drop → "Commit changes". The live site updates in about a minute.
